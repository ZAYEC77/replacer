# jQuery.replacer


# Example of usage

```js
jQuery(function () {
    jQuery('#desktop').replacer([
        [jQuery('#mobile'), '(max-width: 700px)'],
        [jQuery('#super-mobile'), '(max-width: 500px)'],
    ]);
});
```
